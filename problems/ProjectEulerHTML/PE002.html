Solution Explination:

The first part of the code tries to find all Fibonacci's and then add them to a list. This is done primarily by applying the rule of Fibbonacci's sequence Fn+2 = Fn+1 + Fn.

The second part iterates over the list produced by the previous part and checks whether an element is even. If so, the value of that element is added to the sum.
